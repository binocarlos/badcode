# .flow-screenshots/

Scratch folder for screenshots taken while driving Google Flow over the
Playwright/CDP browser (the `animate-slide` / `make-comic` workflows).

When Claude connects Playwright to the logged-in Chrome and grabs a
`browser_take_screenshot`, the `filename` is written **here** instead of the
repo root — e.g. `filename: ".flow-screenshots/gen-progress.png"`.

Everything in this folder is gitignored except this README. The images are
ephemeral debug/progress captures — delete them whenever; they are never
committed.
