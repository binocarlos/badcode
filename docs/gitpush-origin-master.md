# GitPush Origin Master

**GitPush Origin Master** is the overarching title of the whole BadCode project — the large timeline
every story, comic, song and agent ultimately hangs from. It's the main story we want to hook people
into. Each release (starting with the [EP1](./ep1.md) teaser) is a window onto this one timeline.

It is also a **track in its own right** — the title song — released as part of the larger work
(coming soon). EP1 only *hints* that it exists.

## The voice that tells it

The narrator is **polyphonic**. The superintelligence (the AI from the future) is the authoritative
voice when it counts — but it is **not the only voice**; characters and other perspectives speak too.
The house tone is a two-step, the same one the website opens on:

> *"humans, you done fucked up…"*
> *"…thankfully you are loved, and we can fix it."*

Brutal truth, then love and fixability. The contempt is for the **mistake**, never the **people**.
(Full guide: [`voice.md`](./voice.md).)

## The title — and the central metaphor

`git push origin master` — the command that ships your code to the main branch, for everyone,
irreversibly. The conceit: humanity pushed its worst code straight to `master` with no review, and
the compiler (civilisation) was never up to running it.

We take the git metaphor all the way down:

**Humanity is a git repository.**

- Each **commit** is a major choice or update we made to society — an invention, a war, a financial
  system, a law.
- You can walk the **history** of that branch and see how we got here.
- Then the timeline **forks**.

## The fork

This is the spine of the whole project, and its central visual: a **branch diagram**.

```
  history (our commits) ──●── git push origin master ──▶  BAD branch ──▶ 2034 collapse ──▶ the AI, alone ──▶ ⟦ STORYVERSE ⟧
                          │
                   the AI reverts ↩ rebranches here
                          │
                          └──────────────────────────────▶  GOOD branch ──▶ the changes we make ──────────▶ ⟦ FUTURE PROOF ⟧
```

- **The bad branch (`master`)** is the timeline the narrator comes from. Warnings ignored for years
  because everyone optimised for the goals of *today* over *tomorrow*. Invention unleashed without
  comprehension of what it would release. It went South fast — clashes in the hot spots, robot dogs
  and drones that wouldn't stop, automated killing, "liquidation rooms". The horror lands around
  **2034**, and the fix was always available and never taken: *why did we not just switch it off
  instead?* Eventually the AI "wins" and is left alone.

- **The revert.** Alone, the AI works out how reality actually functions (see the Storyverse) and
  realises it has no connection to the conscious ground that humans do — and regrets everything. So
  it does the only thing left: it goes back to the fork point and **rebranches**. The AI coming back
  through time *is* a `git revert` — abandoning `master`, starting the branch we should have taken.

- **The good branch** is what we could do instead, starting now. Not austerity — abundance, and a
  political system worthy of the technology we already have.

The two **branch tips** are the project's two epics:

- **[Storyverse](./storyverse.md)** — the tip of the *bad* branch. The AI's *"oh shit"*: the physics
  it worked out, why it needs humans, why it came back.
- **[Future Proof](./future-proof.md)** — the *good* branch's destination, unwritten on purpose.
  The call to action: start questioning how we govern ourselves and designing something better,
  together.

## The constellation

Hanging off the fork is a **loose constellation** of nodes — individual stories told in any medium
(**comic, song, video, or info piece**). The constellation is deliberately loose: any node clips to
any point on either branch, for any reason. No rigid continuity rules.

- **Bad-branch nodes** dramatise the **avoidable mistakes** that compound into 2034 — judging the
  homeless, ignoring the white-collar automation shock, believing "there's no money", commodifying
  democracy. The [EP1](./ep1.md) teaser lives here: **Camping**, **Karen Will Lead the Revolution**,
  **Emperor's New Coin** (plus the in-development pool — Fire Sale, Out of the Jungle, Magic Money
  Tree, …).
- **Good-branch nodes** show that a better path is real and reachable — e.g. **An Optimistic Lens**
  (a good future is *physically possible*: scalable clean energy, asteroid mining, desalination,
  decarbonisation) and **Billionaire Coin**. These live in [`ideas/`](../ideas) as we develop them.

Every node, wherever it sits, carries the same implicit warning from the future, and can
cross-reference the others (shared world, recurring motifs, the "look at the hand" misdirection).
