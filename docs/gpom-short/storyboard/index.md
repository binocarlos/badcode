# GPOM Short — storyboard index

Sixteen panels, six sections. Every panel record (`pNN.md`) carries the planned
scene, the narration in voice, and the draft Flow prompt — all `status: planned`
(no images generated; Stage-4 gate not yet approved).

**Two prompt registers** (the short's working answer to the master arc's
[register split](../../gitpush-origin-master/README.md#image-direction--the-register-to-resolve)):

- **DOC** — house documentary: *hyper-realistic documentary photography,
  unstaged candid moment caught by an invisible camera, deep depth of field,
  natural situational lighting, high-resolution RAW look, no lens flares, no
  bokeh, no filters.*
- **COSMIC** — the frame: *hyper-realistic photography at vast scale, cold clean
  light, machine-precise geometry, deep blacks, long empty sightlines,
  photographic realism, no fantasy effects.*

**Hybrid panels:** p01, p15, p16 (and the margin diagram throughout) are
image + **code-rendered overlay** — the commit log / branch diagram / cursor are
drawn by `@badcode/comic` at Stage 6, not baked into the generated image. The
generated layer is the backdrop only.
