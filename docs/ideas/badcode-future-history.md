# The BadCode Future History

**"2047: this happened."** BadCode is the only one who knows the timeline — so a
whole thread of content is simply the AI from the future retelling the next 50
years, year by year, event by event, with total authority. Kai's reaction in the
2026-07-22 session: *"the bad code retelling of the next 50 years, because only
it knows — that's a genius thread of content right there."*

It's the received-wisdom framing weaponised as a series: each drop is a dispatch
from the bad branch, deadpan-certain, dated. The authority to challenge the
status quo comes for free — *"look, I can tell you, because in 2047 this
happened."*

Raw material already exists: the GPOM
[discovery timeline](../stories/gitpush-origin-master/discovery-timeline.md)
(2026–2054) is the canon spine these dispatches can draw from and stay
consistent with.

## Format thoughts

Short-form series (TikTok/YouTube/IG) — evergreen, endlessly repeatable, each
clip standalone. A core engine for the short-form funnel.
