# Billionaire Coin

Rich people want to compete with each other over "who is the best," and right now the only
way to do that is to hoard money. So swap them the money for **billionaire coin** — a rare
coin fixed to a billion dollars. The billionaire with the most coins wins, and we use the
actual money to build hospitals.

The egos get to feel superior, and the public will actually like them — because this is a
gamified "get the rich to pay tax."

Already claimed by the good branch: this is one of the named policy-fleet nodes in
[Future Proof](../stories/gitpush-origin-master/future-proof.md) (GPOM's good-branch epic), so
development likely happens there rather than as a standalone story.
