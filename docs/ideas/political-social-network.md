# Political Social Network

A place where ideas are debated, built so you can actually see the shape of public opinion.
Videos on IPFS/Filecoin; Reddit-style, but LLMs sort the videos into "views" — each topic
holds a number of distinct views, and the best video for each view gets upvoted. The
result: a human cross-section of what everyone actually thinks, instead of one algorithmic
winner.
